# BATJTMD 名企考题

## 语义化标签
- div（盒子，结构）+css（样式）+JS(行为) 页面构建的基础
- 可读性问题，不好维护
   div+css 解决页面构建，但是可读性差
- 对搜索引擎不友好
   SEO 对搜索引擎友好
   search engine optimization 搜索引擎优化
   语义化标签，让搜索引擎更好地分析网页
   腾讯是流量提供商（把流量喂给京东，美团。。。）
   阿里是流量吞金兽（流量流向阿里）
- 结构语义化标签
   header 头部
   nav 导航
   main 主要
   article 文章
   aside 侧边
   section 区块
   footer 底部
